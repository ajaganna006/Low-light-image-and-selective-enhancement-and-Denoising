## 🎛️ **How to Use**

### **Website Zoom Controls (Left Panel)**
1. **Zoom Out (-)**: Make entire website smaller to see more
2. **Reset (100%)**: Return to normal size
3. **Zoom In (+)**: Make website larger for detail work

### **Image Zoom Controls (Right Panel)**
1. **Zoom Out (-)**: Make images smaller
2. **Reset (100%)**: Normal image size
3. **Zoom In (+)**: Make images larger for detail inspection

### **Keyboard Shortcuts**
- **Ctrl + Plus**: Zoom website in
- **Ctrl + Minus**: Zoom website out
- **Ctrl + 0**: Reset website zoom
